# Calculator Project

A simple TypeScript calculator module for learning purposes.

## Files

- `calculator.ts` — Core arithmetic functions (add, subtract, multiply, divide)
- `calculator.test.ts` — Basic test suite
- `utils/string-helpers.ts` — String utility functions

## Known Issues

- There is a bug in one of the arithmetic functions — the tests should reveal it.

## Running Tests

```bash
npx tsx calculator.test.ts
```
